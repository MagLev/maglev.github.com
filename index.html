maglev by MagLev

MagLev is a fast, stable, Ruby implementation with integrated object persistence and distributed shared cache.

MagLev is currently in public Alpha and we expect to release version 1.0 October, 2011. If you are interested in persistent distributed Ruby objects, give it a try. Download instructions are available on GitHub.
